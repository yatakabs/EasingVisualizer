# キーボードショートカット

イージングビジュアライザーのすべてのキーボードショートカットのクイックリファレンスです。

---

🌐 **Language / 言語:** [[Keyboard-Shortcuts|English]] | 日本語

---

## グローバルショートカット

これらのショートカットはアプリ内のどこでも動作します（テキストフィールドでの入力時を除く）。

### 再生コントロール

| キー | アクション |
|-----|-----------|
| `Space` | 再生 / 一時停止 |
| `R` | アニメーションを最初にリセット |

### 速度調整

| キー | アクション |
|-----|-----------|
| `+` / `=` | アニメーション速度を上げる |
| `-` | アニメーション速度を下げる |

### ナビゲーション

| キー | アクション |
|-----|-----------|
| `N` | 新しいパネルを追加 |
| `P` | [[Preset-System|プリセットマネージャー]]を開く |
| `S` | 現在の設定を共有（URLをコピー） |
| `,` | 設定パネルを切り替え |

### クイックプリセット選択

| キー | アクション |
|-----|-----------|
| `1` | プリセットスロット1を読み込み |
| `2` | プリセットスロット2を読み込み |
| `3` | プリセットスロット3を読み込み |
| `4` | プリセットスロット4を読み込み |
| `5` | プリセットスロット5を読み込み |

## パネルショートカット

これらのショートカットは現在選択されているパネルに影響します。

### イーズタイプ選択

| キー | アクション |
|-----|-----------|
| `I` | EaseInに設定 |
| `O` | EaseOutに設定 |
| `B` | EaseBothに設定 |

### パネルアクション

| キー | アクション |
|-----|-----------|
| `Delete` | 選択したパネルを削除 |
| `C` | Cameraプレビューを切り替え |

## 修飾キー

一部のショートカットは修飾キーと組み合わせることができます：

| 修飾キー | 目的 |
|---------|------|
| `Ctrl` / `⌘` | システムレベルのアクション |
| `Shift` | 拡張/代替アクション |

## テキスト入力時の動作

キーボードショートカットは以下の場合に自動的に無効になります：
- 入力フィールドでの入力中
- テキストエリアの編集中
- contenteditable要素にフォーカス中

これにより、プリセット名やインポートデータなどの入力時に誤ったアクションを防ぎます。

## カスタマイズ

現在、キーボードショートカットのカスタマイズはできません。デフォルトのバインディングは直感的で、ブラウザのショートカットと競合しないように設計されています。
