# Changelog

Version history for EasingVisualizer.

## Current Version

### v1.0.0 (December 2024)

Initial public release with full feature set:

#### Features
- **16+ easing functions** - Complete set of standard easing curves
- **4 preview types** - Glow, Graph, Camera, Value visualizations
- **Comparison mode** - Side-by-side function comparison
- **ScriptMapper mode** - Beat Saber integration
- **Preset system** - Save and load configurations
- **URL sharing** - Share configurations via URL
- **Mobile support** - Fully responsive design
- **Keyboard shortcuts** - Power user navigation

#### Technical
- Built with React 19, TypeScript, Vite
- Three.js for 3D camera preview
- Radix UI (shadcn/ui) components
- Tailwind CSS styling
