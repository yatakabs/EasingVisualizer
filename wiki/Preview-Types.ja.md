# プレビュータイプ

イージングビジュアライザーは、イージング関数の動作を理解するための5種類の可視化タイプを提供しています。

---

🌐 **Language / 言語:** [[Preview-Types|English]] | 日本語

---

## 概要

| プレビュータイプ | 目的 | 最適な用途 |
|-----------------|------|-----------|
| **Glow** | 視覚的な強度 | クイックな視覚比較 |
| **Graph** | 数学的カーブ | 関数形状の理解 |
| **Camera** | 3D移動 | Beat Saberカメラパス |
| **Value** | 数値表示 | 正確な値の確認 |
| **Easing Compare** | 並列表示 | イーズバリエーションの比較 |

## Glowプレビュー

Glowプレビューは、イージング関数の出力に基づいて明るさが変化する円形要素を表示します。

### 特徴

- **放射状のグロー効果** - パネル中央にソフトなグラデーション
- **リアルタイムの明るさ** - 強度は出力値（0-1）に直接マッピング
- **カラーカスタマイズ** - 各関数に固有のアクセントカラー

### ユースケース

- 複数の関数の視覚的特性をすばやくスキャン
- スムーズなトランジションと急なトランジションの違いを理解
- LED/ライトアニメーションのプレビュー

### 技術的詳細

- 出力値0 = 最小輝度
- 出力値1 = 最大輝度
- 一貫した彩度のためにOKLCH色空間を使用

## Graphプレビュー

Graphプレビューはイージング関数カーブの2Dプロットを表示します。

### 特徴

- **X軸**：時間/入力（0から1）
- **Y軸**：出力値（0から1）
- **アニメーションカーソル**：カーブ上の現在位置を表示
- **参照線**：グリッドで重要なポイントを可視化

### グラフの読み方

| カーブ形状 | 意味 |
|-----------|------|
| 急な傾斜 | 急速な変化 |
| 緩やかな傾斜 | ゆっくりした変化 |
| S字カーブ | 加速して減速 |
| オーバーシュート | 0-1範囲を超える |

### ユースケース

- 数学的動作を理解
- カーブ形状を正確に比較
- オーバーシュートの特定（Back、Elastic関数）
- 教育目的

### オプション

- 生の出力とガンマ補正出力の切り替え
- 参照グリッドの表示/非表示
- 関数式の表示

## Cameraプレビュー

Cameraプレビューはイージング関数に基づいてカメラ移動を示す3Dシーンをレンダリングします。

### 特徴

- **3Dビューポート** - シーンの透視図
- **アニメーションカメラ** - 位置がイージング関数を使用して補間
- **参照ジオメトリ** - グリッドフロア、座標軸
- **アバタープレースホルダー** - カメラフォーカスの視覚的ターゲット

### カメラ設定

| 設定 | 説明 |
|-----|------|
| **開始位置** | カメラの開始点（X, Y, Z） |
| **終了位置** | カメラの終了点（X, Y, Z） |
| **FOV** | 視野角 |
| **アスペクト比** | ビューポートの寸法 |

### ユースケース

- Beat Saberカメラパスのプレビュー
- 3Dアニメーションカーブの可視化
- カメラ移動の滑らかさをテスト
- [[ScriptMapper-Mode.ja|ScriptMapperモード]]ワークフロー

### 技術的詳細

- Three.jsでレンダリング
- 目標フレームレート60fps
- 左手系と右手系の座標系に対応

## Valueプレビュー

Valueプレビューは数値の入力値と出力値を表示します。

### 特徴

- **入力値** - 現在の正規化時間（0-1）
- **出力値** - イージング関数の結果
- **リアルタイム更新** - アニメーション中に値が変化
- **高精度** - 複数の小数点以下桁を表示

### 表示

```
Input:  0.500
Output: 0.750
```

### ユースケース

- 特定の時点での正確な値を確認
- アニメーションの問題をデバッグ
- 教育 - 入力が出力にどのようにマッピングされるかを確認
- アクセシビリティ - 非視覚的な検証

## Easing Compareプレビュー

Easing Compareプレビューは1つの関数の3つのイーズタイプを並べて表示します。

### 特徴

- **3つのサブパネル** - EaseIn、EaseOut、EaseBoth
- **同期アニメーション** - 3つとも一緒に実行
- **同じ関数** - 1つの関数のバリエーションを比較

### レイアウト

```
┌──────────┬──────────┬──────────┐
│  EaseIn  │ EaseOut  │ EaseBoth │
└──────────┴──────────┴──────────┘
```

### ユースケース

- In/Out/Bothの違いを理解
- 適切なイーズタイプをすばやく選択
- 加速と減速を比較
