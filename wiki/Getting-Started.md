## Basic Usage

### 1. Choose Your Mode

The app offers two main visualization modes:

| Mode | Best For |
|------|----------|
| **[[Comparison-Mode|Comparison Mode]]** | Comparing easing functions side-by-side |
| **[[ScriptMapper-Mode|ScriptMapper Mode]]** | Beat Saber camera path visualization |

### 2. Select Easing Functions

1. Click on a panel to select it
2. Use the function selector dropdown to choose an easing function
3. Select the ease type: **EaseIn**, **EaseOut**, or **EaseBoth**

### 3. Control Playback

- **Play/Pause** - Start or stop the animation
- **Speed** - Adjust animation speed (0.1x to 10x)
- **Manual Mode** - Scrub through the animation manually using a slider

### 4. Customize the View

- **Preview Type** - Switch between Glow, Graph, Camera, Value, or Easing Compare views
- **Panel Scale** - Adjust the size of preview panels
- **Gamma Correction** - Apply output filtering for different visual effects

## Saving Your Work

### Save as Preset

1. Configure your panels and settings
2. Click **Save Current State** in the Preset Manager
3. Enter a name for your preset
4. Click **Save**

### Share via URL

1. Click the **Share** button in the toolbar
2. Copy the generated URL
3. Share with others - they'll see a preview banner to apply your configuration
