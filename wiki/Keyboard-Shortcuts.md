# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Easing Visualizer.

## Global Shortcuts

These shortcuts work anywhere in the app (except when typing in text fields).

### Playback Controls

| Key | Action |
|-----|--------|
| `Space` | Play / Pause animation |
| `R` | Reset animation to start |

### Speed Adjustment

| Key | Action |
|-----|--------|
| `+` / `=` | Increase animation speed |
| `-` | Decrease animation speed |

### Navigation

| Key | Action |
|-----|--------|
| `N` | Add new panel |
| `P` | Open [[Preset-System|Preset Manager]] |
| `S` | Share current configuration (copy URL) |
| `,` | Toggle Settings panel |

### Quick Preset Selection

| Key | Action |
|-----|--------|
| `1` | Load preset slot 1 |
| `2` | Load preset slot 2 |
| `3` | Load preset slot 3 |
| `4` | Load preset slot 4 |
| `5` | Load preset slot 5 |

## Panel Shortcuts

These shortcuts affect the currently selected panel.

### Ease Type Selection

| Key | Action |
|-----|--------|
| `I` | Set to EaseIn |
| `O` | Set to EaseOut |
| `B` | Set to EaseBoth |

### Panel Actions

| Key | Action |
|-----|--------|
| `Delete` | Remove selected panel |
| `C` | Toggle Camera preview |

## Modifier Keys

Some shortcuts can be combined with modifier keys:

| Modifier | Purpose |
|----------|---------|
| `Ctrl` / `⌘` | System-level actions |
| `Shift` | Extended/alternate actions |

## Text Input Behavior

Keyboard shortcuts are automatically disabled when:
- Typing in an input field
- Editing a textarea
- Focused on a contenteditable element

This prevents accidental actions while entering preset names, import data, etc.

## Customization

Currently, keyboard shortcuts cannot be customized. Default bindings are designed to be intuitive and non-conflicting with browser shortcuts.

## Tips

1. **Learn the essentials first** - `Space`, `N`, and `I`/`O`/`B` cover most needs
2. **Use number keys** - Quick preset loading speeds up workflow
3. **Remember C for Camera** - Quickly toggle 3D preview on selected panels
4. **Settings with comma** - Same as many apps (VS Code, etc.)
